## 一、品牌宣传

### 1. 为什么我带孩子去巴厘岛上课-推文.docx / .pdf
- **用途**：品牌故事推文，面向家长端传播
- **内容概要**：以一位母亲的第一人称视角，讲述从「为什么暑假总是千篇一律」的反思开始，引出森林学校（Forest School）、华德福（Waldorf）、永续生活（Permaculture）三大教育理念，介绍巴厘岛亲子游学营的课程总监 Carolina Leme 及其国际团队，呈现7天行程安排和「分合式」亲子设计（Day 1-2亲子共同 → Day 3-4分开成长 → Day 5-7再次合体），最后以孩子回家后的真实改变收尾
- **字数**：约3000字
- **适用场景**：公众号推文、官网品牌页、朋友圈长文

### 2. 为什么我带孩子去巴厘岛上课-绘本.pdf
- **用途**：品牌故事绘本版，适合亲子共读
- **内容概要**：将推文内容转化为绘本风格，图文并茂，适合向4-8岁儿童介绍游学营内容
- **页数**：多页图像为主
- **适用场景**：招生说明会展示、线上绘本传播、客户到访阅读材料

### 3. 巴厘岛亲子游学营-品牌宣传片-V2.mp4
- **用途**：60秒品牌宣传视频
- **文件大小**：50.5 MB

- **适用场景**：视频号、抖音、小红书、招生说明会开场

## 二、产品方案

### 4. 巴厘岛亲子游学营（师资介绍）.pdf
- **用途**：师资团队详细介绍
- **核心内容**：
  - **Carolina Leme**（课程总监/首席教师）：20余年跨文化儿童教育经验，职业经历横跨墨西哥、危地马拉、巴西和巴厘岛；持英国森林学校培训师资格、华德福教育基础认证、永续生活设计认证；巴厘岛森林学校联合创始人
  - **外籍主教老师**：美国学士，2015年北京任教，2019年「年度教师」，5年以上中国学生教学经验
  - **双语主教老师**：国际教育背景，PBL项目协助，双语教学衔接
  - **全英助教 ×2**：儿童陪伴与户外安全保障
  - **运营支持团队**：司机+翻译+统筹+应急，共7人
- **适用场景**：客户咨询发送、合作伙伴背书信、网页「师资」板块素材

### 5. 巴厘岛亲子营安全保障白皮书.pdf
- **用途**：面向幼儿园园长和合作渠道的完整方案书
- **核心内容**：
  - Part 1「我们是谁」：巴厘岛在地办学团队介绍
  - Part 2「为什么与学前从业者同频」：森林学校+华德福+永续生活三大体系的融合，与幼儿园日常实践的衔接对照
  - Part 3「7天行程」：每日详细安排，含 Day 3 海洋守护者和 Day 4 森林守护者的完整课程教案（欢迎圈→故事→探索→游戏→手工→反思圈），以及「分合式」设计的差异化价值说明
- **适用场景**：园长沟通、渠道招商、投资人展示

### 6. 巴厘岛亲子营安全保障手册.pdf
- **用途**：内部全维度安全管控操作手册
- **核心内容**：
  - 五大安全原则：安全第一、预防为主、儿童优先（双确认制）、全员负责（实时通报）、底线思维（场景预案全覆盖）
  - 行前准备：证件核查（出发前30天）、保险购买（保额50万+，出发前7天）、健康信息收集（出发前14天）、线上安全说明会（出发前7天含儿童专场）
  - 出入境与交通：VOA落地签流程、KIR合规车辆（车龄<5年）、安全带/儿童座椅/驾驶时间规范
  - 住宿与餐饮：努沙杜瓦五星酒店标准、儿童专属菜单+过敏源标注
  - 涉水活动：分级管理制度
  - 应急预案：随队药箱、本地医院绿色通道、24小时联络人
- **适用场景**：内部培训、家长行前说明会、安全审计备查

### 7. 巴厘岛亲子营每日签到安全管控表（范本）.pdf
- **用途**：每日安全签到与追溯记录模板
- **内容概要**：逐日签到的标准化表格，涵盖儿童出勤、身体状况、活动参与确认、安全事项勾选等维度
- **适用场景**：营期执行中的日常管理

### 8. 渠道合作手册-面向园长.docx / .pdf
- **用途**：面向幼儿园园长的渠道合作招商方案
- **内容概要**：合作模式、阶梯分润机制、专属营期定制方案、园所差异化增值服务说明
- **适用场景**：幼儿园渠道拓展、教育展会资料、商务洽谈
